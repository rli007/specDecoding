Recommended speculative decoding pairs
=====================================

The target and assistant should use the same tokenizer/vocab for this repo's
first-principles implementation.

Llama 3 family
--------------

Base continuation pair:

target:    meta-llama/Meta-Llama-3-8B
assistant: meta-llama/Llama-3.2-1B

Instruction/chat pair:

target:    meta-llama/Meta-Llama-3-8B-Instruct
assistant: meta-llama/Llama-3.2-1B-Instruct

Notes:
- These are gated Hugging Face repos. Accept the Meta license and run
  `huggingface-cli login` before loading them locally.
- Llama 3 8B is large. CPU-only inference can be slow and memory-heavy.
- Use the Llama 3.2 1B assistant rather than TinyLlama 1.1B for Llama 3 targets;
  TinyLlama is Llama 2-tokenizer based and is not compatible with this simple
  same-vocab speculative decoder.
